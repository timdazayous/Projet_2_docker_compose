# Guide de Contribution

Merci d'envisager de contribuer à ce projet !

## Comment contribuer

1. **Forkez** le dépôt.
2. Créez une **branche** pour votre fonctionnalité (`git checkout -b feature/ma-fonctionnalite`).
3. **Committez** vos changements (`git commit -m 'Ajoute une super fonctionnalité'`).
4. **Poussez** la branche (`git push origin feature/ma-fonctionnalite`).
5. Ouvrez une **Pull Request**.

## Règles de développement

- Écrivez des tests unitaires pour le nouveau code.
- Assurez-vous que le linting (Ruff) passe.
- Ne commitez JAMAIS de secrets ou de données sensibles (vos commits sont scannés par Gitleaks).
